When user has a YouTube video with the transcript open, this will summarize, analyze bias, and provide counterpoints. Fact check and custom instructions are available.

  • The transcript is scraped from the tab (no automated way is possible rn, currently google is blocking all ips that use these tools)
  • The video ID is taken from the open tab, and that is sent to an AWS Lambda function which gets the video's title and duration from the YouTube Data API.
  • The transcript is sent to the ChatGPT4o-mini API via an AWS Lambda function.
  • ChatGPT is instructed to summarize, provide bias analysis, and return a suitable Google search query
  • Chat's response is parsed for the query, and that query is sent to a Google custom search API via AWS Lambda function.
  • Data is then parsed for the results tab.

    Overview:
    Yt-sum is a YouTube Video Summarizer & Analyzer extension designed to summarize YouTube videos.

    Features:
  • Instant Summaries: Get a concise summary of any YouTube video with a single click. Perfect for quickly understanding the core message without watching the entire video.
  • Bias Detection: The AI can analyze the transcript for biases, helping you identify any potential slant or perspective within the content.
  • Google Search Integration: Automatically generate suggested Google search queries based on the videoâ€™s content to explore related perspectives and counterpoints.
  • Time-Saving: Ideal for students, researchers, and busy professionals who need to consume information quickly.
  • Fact check option
  • Custom instruction option
  • Privacy-Focused: The extension processes the video transcripts locally and securely, ensuring your data remains private.

    • How It Works:
  • Add the extension to your Chrome browser.
  • Navigate to a YouTube video you want to summarize or analyze.
  • Open transcript
  • Click the Extension Icon
  • Choose options
  • Extension will query ChatGPT 4o-mini for a summary, bias analysis, fact check and custom instructions if requested
  • The summary and analysis are displayed in a new tab.
