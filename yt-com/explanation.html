Non AI Generated Gobbledegook Info for Nerds:

When user has a YouTube video open, this will summarize/analyze the comments section according to the user instructions.

  • The video ID is taken from the open tab, and that is sent to an AWS Lambda function which gets the video's title from the YouTube Data API.
  • The comments section is fetched via Youtube Data API via AWS Lambda function.
  • After the user chooses their options, the comments are sent to ChatGPT4o-mini API via an AWS Lambda function.
  • Returned data is then parsed for the results tab.
